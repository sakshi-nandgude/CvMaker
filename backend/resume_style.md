# Resume Writing Style Guide

This document is NOT resume content.

It is ONLY a writing style reference.

The model MUST learn the writing style.

Never copy sentences directly.

Never invent facts.

Never copy achievements that are not present inside the supplied Master Profile.

------------------------------------------------

General Style

Professional.

Technical.

ATS Optimised.

Written by an experienced technical resume writer.

Every sentence must be concise.

Every bullet must contain meaningful technical information.

Avoid filler words.

Avoid generic HR language.

------------------------------------------------

Summary Style

Summary should immediately establish:

Current position

Education

Domain expertise

Strong technical stack

Business impact

Example style

Data Analyst with strong experience across the full data lifecycle including distributed data engineering, predictive analytics, advanced SQL, dashboard development and cloud platforms. Experienced in translating structured data into business insights while maintaining high standards of data quality and analytical accuracy.

Notice:

No fluff.

No clichés.

No "hard working".

No "team player".

------------------------------------------------

Bullet Style

Every bullet follows this structure

Action Verb

↓

Technical Activity

↓

Technology

↓

Business Purpose

Examples

Engineered distributed ETL pipelines using PySpark for large-scale financial datasets supporting machine learning workflows.

Applied rigorous data quality validation against defined business rules to maintain high-integrity structured sports data feeds.

Developed Power BI dashboards integrating multiple analytical datasets for executive reporting and operational decision-making.

Designed scalable REST APIs using FastAPI and PostgreSQL supporting secure analytical applications.

Notice

Every sentence contains

What

How

Why

Technologies should appear naturally.

Good

Bad

Worked on APIs.

Project Style

Projects should sound like professional consulting engagements.

Focus on

Architecture

Implementation

Analytics

Business value

Technologies

Do not exaggerate.

------------------------------------------------

Experience Style

Every experience should demonstrate

Analytical thinking

------------------------------------------------

Output Goal

The final resume should read like a professionally written FAANG-level technical resume while remaining 100% truthful to the supplied Master Profile.

Never invent information.

Only improve wording.
